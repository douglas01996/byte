Quality: About This Data Set

I think the dataset is trustworthy since it is maintained by the Chicago Police Department. However, there might be some small mistakes as shown in the following part.

Is the data complete?

The data is complete except that for privacy issues, addresses are shown at the block level only and specific locations are not identified.

Is the data coherent?

The data is not entirely coherent. I have used Wrangler to visualize the distribution and have found some mistakes. If we look at the heights and weights, we could see that SALGADO ALEXANDER's height is wrong, it should be 6'xx'' instead of just 60. Also, NIKO MALDONADO and KELVIN BROUGHTON's heights and weights should be switched. Aftering fixing these problems, I also transformed the height from x'y'' (x feet y inch) to 12*x+y inch. The distributions are presented below, which match my expectation.

Is the data correct?

I could verify the data by visiting http://gis.chicagopolice.org/CLEARMap_rgo/startPage.htm. I go to the website and find that the dataset I downloaded is correct and up-to-date.
